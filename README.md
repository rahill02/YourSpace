# YourSpace

A full-stack note-taking web application.

## Tech Stack
- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Database: MongoDB Atlas

## Features
- Create, edit, delete notes
- User authentication
- Clean UI

## Setup

### Backend
- cd backend
- npm install
- npm run dev

### Frontend
- cd frontend
- npm install
- npm run dev
